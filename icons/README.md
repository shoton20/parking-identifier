# App Icons

This folder should contain the following PNG icons for the PWA:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## How to Create Icons

You can create icons using any of these methods:

### Option 1: Online Icon Generator
1. Visit https://www.pwabuilder.com/imageGenerator
2. Upload a square image (at least 512x512px)
3. Download the generated icons
4. Place them in this folder

### Option 2: Design Tool (Figma, Canva, etc.)
1. Create a 512x512px square design
2. Use the car emoji 🚗 or create a custom parking icon
3. Export in different sizes listed above
4. Place them in this folder

### Option 3: Simple Placeholder (for testing)
For quick testing, you can use a simple colored square:
1. Create a 512x512px image with a blue background (#4A90E2)
2. Add white text "P" in the center
3. Resize to all required sizes
4. Place them in this folder

## Temporary Solution

Until you create proper icons, the app will work but won't be installable as a PWA. The browser will show a warning about missing icons.
